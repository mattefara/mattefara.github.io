Date: dal 05/06/2017 al 23/06/2017 e dal 11/09/2017 al 29/09/2017

In queste settimane ho svolto il periodo di alternanza scuola lavoro presso l'azienda 70Division.

Ho lavorato come web developer sviluppando sia parti di codice sia parti di sito.

Ho sviluppato nuove compenze e ho conoscuito nuovi programmi.
Le conpetenze di base che ho acquisito sono:

  • L'utilizzo di Git
  • L'utilizzo di Pug
  • L'utilizzo di SCSS
  • L'utilizzo del database di Parse
  • L'utilizzo delle $.get di Ajax
  • L'utilizzo delle APi di Google, di Slack e di Facebook places
  • L'utilizzo di Bootstrap

I primi quattro punti mi sono serviti per lavorare a dei piccoli miglioramenti nel loro sito web Blink.

Git, Pug e Scss mi erano nuovi ma durante il periodo del loro utilizzo li ho trovati molto utili. Infatti agevolano in lavoro di web developer.

Git viene utlizzato per fondere le diverse versioni che i vari sviluppatori stanno sviluppando separatamente senza che ci siano perdite di dati, mentre Pug e SCSS sono du linguaggi che agevolanno la scritture dei rispettivi codici HTML e CSS. Infatti Pug è basato su python e non prevede che i tag vengano chiusi ma l'indentazione deve essere corretta mentre SCSS è una versione del CSS semplificata. A sinistra Pug e SCSS e a destra HTML e CSS.

Sia SCSS che Pug vengono gestiti dal terminale e ogni volta che vengono salvati il contenuto del file viene convertito in HTML e CSS.

Invece per quanto riguarda gli ultimi punti della lista li ho utilizzati solamente una volta per fare delle pinccole integrazioni sempre con il sito web.
