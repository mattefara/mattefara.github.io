Corso avanzato di CSS

Ormai è risaputo che il CSS (cascading style sheet) con HTML sono come il pane e il burro: inseparabili. Assieme all' HTML, è infatti lo strumento più importante per lo sviluppo di siti web. Come HTML, il css NON è un linguaggio di programmazione, bensì un linguaggio che definisce lo stile ed il layout delle pagine web. Si, oramai ci sono molti altri liguaggi magari più adatti alla grafica del sito web come per esempio Javascript, Ajax o Php, ma nessuno sarà mai completo e "potente" come il css, anche perchè bisogna comunque saperlo bene prima di cominciare con dei linguaggi di programmaione web.
